# 🔐 Secure Memory Isolation System

## 📌 Overview
This project demonstrates memory isolation in operating systems using C, Node.js, and an interactive frontend.

## 🧠 Features
- Memory allocation and protection
- Unauthorized access simulation
- Interactive RAM visualization
- Process-based simulation (P1, P2)

## ⚙️ Tech Stack
- C (Windows API)
- Node.js (Express)
- HTML, CSS, JavaScript

### ⚙️ Tools & Libraries
- Windows API (`VirtualAlloc`, `VirtualProtect`)  
- Node.js + Express  
- GCC (MinGW)
